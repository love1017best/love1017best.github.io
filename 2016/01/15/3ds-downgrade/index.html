老 3DS（非 LL）10.3 正版降级 9.2 并无卡破解记录
拾与拾柒
 2016-01-15 20:55:20

此破解方法已经过于陈旧，现在想要破解的朋友可以直接使用 B9S，网上可以找到非常详细的教程。

前不久 3DS 全线告破，也想给手上的一台专门用来玩下载版口袋妖怪的老 3DS 来个无卡破解。 DS 时代非常关注破解和自制，但是进入 3DS 时代以后一直没怎么关注过这方面的动向，一直当着正版党，烧录卡都没买过，这次听说终于全线告破还是有些小小的激动，早就想尝试汉化游戏了，无奈之前手上并没有可以破解的机器，于是决定动手一试。折腾了几个小时，最后有惊无险地成功了，遂将全过程记录一下，方便以后查阅。

以下所有步骤都只针对日版小3DS（非 LL，非 new 3DS）。理论上老 3DS LL 也适用，但是我并没有测试过。

【第一阶段】降级

  1. 确认系统版本为 10.3-28j ；
  2. 可以准备一张 SD 卡，我因为原来机器用的是一张 16G 的卡，就直接沿用了；
  3. 如果需要保留购买的下载版游戏、存档和主题等内容，请备份 SD 卡全部内容（因为我想保留正版下载版游戏和存档以及主题等内容）至电脑；
  4. 下载 http://pan.baidu.com/s/1i4cky8X （密码：ogsm），将所有内容复制到 SD 卡根目录下；
  5. 插卡开机，切换主题至红色，再切换回白色默认主题（如果开了自动换主题，最好关闭），打开浏览器，点下屏的扳手图标，选择初始化；
  6. 输入网址： http://yls8.mtheall.com/spider28hax.php ，等待下屏依次变成黄、灰、绿、红后，自动进入 HBL (The Homebrew Launcher) ，如果进入失败或卡在某一步，长按电源键强行关机，从 5 开始重做，直至成功为止；
  7. 进入 HBL 以后，选择 menuhax_manager v2.0 一项，进入黑底白字界面，按 A 键选择“ Install ”，看到 successful 字样后表明 menuhax 安装成功，长按电源键关机；
  8. 开机同时按住 L 键，会看到上屏花屏，下屏依次切换颜色，待进入 HBL 菜单（蓝色波纹界面）后即可松手，选择 ftBRONY 一项，进入后等待 10 秒以上，按 B 键，退出回到 HBL 菜单；
  9. 在 HBL 中选择 sysUpdater ，进入后按 Y 键运行，待所有 CIA 安装完毕后，等待 10 秒，机器自动重启，降级至 9.2 完成；如果此时未重启或无反应，请等待 10 秒以上手动关机，重新进入系统，如果看到版本变为 9.2 则同样降级成功；请不要在 CIA 安装过程中关机，否则机器变砖！
【第二阶段】无卡破解
  1. 首先确认系统版本为 9.2 ；
  2. 将 SD 卡格式化为 FAT32 格式，如果要保留以前的内容，务必备份（主要是备份 Nintendo 3DS 文件夹），不要初始化 3DS 本体；
  3. 下载 Launcher.dat 文件（链接：http://pan.baidu.com/s/1o6XW7RO 密码：fbt9），置于 SD 卡根目录，插入 3DS 开机；

  6. 扫描后进入红色龙的 GW 菜单，选择 backup sysNAND ，待备份完成后会回到红龙界面，长按电源键关机；
  7. 将此时 SD 卡内的 NAND.bin 备份好，供今后手贱升级了重新降级使用；
  8. 重复 4、5 步，选择 FORMAT EMUNAND ，待完成后会回到红龙界面，长按电源键关机；
  9. 接下来的步骤请全程联网；
  10. 确认 SD 卡的可见容量少了 1G ，说明虚拟系统制作成功；
  11. 下载 http://pan.baidu.com/s/1o7kNI0E（密码：bp9r），将所有内容置于 SD 卡根目录；
  12. 开机，切换一次红色主题后，再切换回默认白色主题，打开浏览器，初始化浏览器，输入网址：http://yls8.mtheall.com/sliderhax.php ，待下屏出现一条短横线，用触摸笔点击后，会进入 HBL 菜单（之前见过的蓝色波纹界面）；
  13. 如果失败，请不断重复第 12 步，每个步骤都不要省略，直至成功进入 HBL 为止；
  14. 进入 HBL 以后，选择 menuhax_manager v2.0 一项，进入黑底白字界面，按 A 键选择“ Install ”，看到 successful 字样后表明 menuhax 安装成功，长按电源键关机；
  15. 开机同时长按 L ，直至进入 HBL ，选择 rxTools 一项；
  16. 待 rxTools 安装完毕后会自动进入虚拟系统，查看系统版本，如果为 RX-E 9.2.0-20J 说明成功，退出设置后会自动回到真实系统，这是正常现象，今后破解完成后也是如此；
  17. 关机，重新开机同时长按 L 进入 HBL ，选择 rxTools ，下屏会先变红，红屏消失时立刻开始长按 L 键，会进入 rxTools 菜单（蓝色界面，如果不按 L 键，则会直接进入虚拟系统）；
  18. 用左右键选择 ADVANCE 一项，再选择 INSTALL FBI ，按 Y 键选择 emuNAND ，再按 B 选择 check TMD only，此时会显示 Region 和 Version of TMD 两项信息，记录下来后面有用；
  19. 下载 FBI 文件包（链接 http://pan.baidu.com/s/1eQSNm8q 密码：yr3h），根据刚才的 Region 和 Version of TMD 找到对应文件夹，将其中的两个文件复制到 SD 卡根目录；
  20. 下载文件：http://pan.baidu.com/s/1bnKLLmv （密码：m2ao），放到 SD 卡中 Nintendo 3DS\xxxxxxxxxxx\xxxxxxxxxxx\ 路径下（每台机器不同的两串字符）；
  21. 下载文件：http://pan.baidu.com/s/1jHvfvBw （密码：w0wc），置于 SD 卡根目录；
  22. 开机，进入真实系统，进入本体设定，进入左下角数据管理，选择 3DSdate 管理，选择第一项，报错之后点击“初期化する”，完成后关机；
  23. 重复 17 进入 rxTools 菜单；
  24. 选择 ADVANCE ，再选择 INSTALL FBI ，按 Y 键选择虚拟系统，再按 Y 选择注入 FBI ，待完成后回到 rxTools 菜单，选择第一项，会进入虚拟系统，选择黄色叹号项，若看到黑底白字的 FBI 界面，则说明之前的操作成功；
  25. 此时下屏为文件管理界面，移动光标至 TOOLS_CIA ，安装其中的两个文件 DEVMENU 和 BBM ，完成后按 HOME 键回到虚拟系统首页，会看到新软件提示，就是刚才安装的两个工具；
  26. 关机拔卡，在无 SD 卡的情况下初始化 3DS 本体，机器重启过程中看见屏幕上出现 3DS LOGO ，强行关机；
  27. 查看 SD 卡，若有这几个文件则删除： ropbinpayload_menuhax_JPN15360_old3ds.bin, screenshots_raw.bin, webkithax_tmp.bin；
  28. 将 SD 卡插入，开机，经过初始化设置进入真实系统后关机；
  29. 重复第 12 步：开机，切换一次红色主题后，再切换回默认白色主题，打开浏览器，初始化浏览器，输入网址：http://yls8.mtheall.com/sliderhax.php ，待下屏出现一条短横线，用触摸笔点击后，会进入 HBL 菜单（之前见过的蓝色波纹界面）；
  30. 至此免卡破解完成，今后开机长按 L 键进入 HBL ，选择 rxTools 即可进入虚拟系统，虚拟系统可以直接通过 eshop 升级至最新的 10.3-28j 系统，如果不需要恢复原有的数据，至此破解结束；
  31. 今后所有游戏都在虚拟系统中玩，在网络上找到 CIA 格式的游戏，放入 SD 卡中，进入 DevMenu 选择 import 选项，找到 CIA 文件安装即可，安装之后可以删除 SD 卡中的 CIA 文件；
  32. 如果要恢复购买的下载游戏、存档及主题，请继续；
  33. 重新将最初备份的 Nintendo 3DS 文件夹中的随机乱码个人游戏数据文件夹放回 SD 卡并替换已有的文件夹（ Nintendo 3DS\xxxxxxxxxxx\xxxxxxxxxxx\ 文件夹内的内容，其中应该是有 DevMenu_2x 和 BBM 两个工具）后，虚拟系统成功读取到了降级之前的所有本地内容（我自己的包括两款下载版正版以及若干 eshop 试玩版游戏还有几款主题），如果 DevMenu_2x 和 BBM 找不到了，重新在虚拟系统进入 3DS 的黄色叹号项安装即可（参考第 25 步）；
  34. 此时我遇到了最崩溃的一件事：打开口袋 OR 正版下载版后提示存档不支持（当时我就蒙逼了，毕竟是几百小时 + 600 多图鉴的存档啊），但是奇怪的是口袋 X 正版下载版能正常进入游戏且存档完好，不过好在找到了解决办法；
  35. 造成问题的原因是口袋妖怪游戏的存档验证机制，简单来说就是避免复制存档的，实际上此时游戏的存档只是被锁定了而并没有被删除（千万不要手贱新开存档），所以使用神器 SDF ( SaveDataFiler ，戳此下载：SaveDataFiler)即可解除锁定，安装的过程同样是将 CIA 文件放入 SD 卡，进入虚拟系统的黄色叹号项找到对应项目安装即可；
  36. 在虚拟系统中找到口袋 OR 游戏，按一次 SELECT 键，然后进入 SDF ，找到选项或者直接找到游戏对应的代码，按下 R + X 解除锁定，之后就可以继续享受游戏啦！顺便记录一下查到的 3DS 四作口袋妖怪正统作的 SDF 内代码：
口袋妖怪 X —— 0055D
口袋妖怪 Y —— 0055E
口袋妖怪 始源蓝宝石—— 011C5
口袋妖怪 终极红宝石—— 011C4
至此整个【降级 + 破解】过程结束，可以享受汉化游戏了，破解后也有很多有意思的内容比如玩 GBA、GBC 游戏以及各种自制软件、自制系统等，今后再慢慢研究！
